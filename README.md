# API-RESTful-PHP
API RESTful with PDO

## ATTENTION
Rename the file `a.htaccess` to `.htaccess`

## USE
This API works with a friendly URLs

### GET
Get all bookts: 
```
https://server/api/books
```
Get specific book: 
```
https://server/api/books/5
```
### POST
Insert a book: 
```
https://server/api/books
```

### PUT
Update a book: 
```
https://server/api/books/5
```

### DELETE
Delete a book: 
```
https://server/api/books/5
```
